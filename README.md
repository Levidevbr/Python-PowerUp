# 🐍 Python PowerUp

Projeto desenvolvido durante meus estudos de Python com o objetivo de praticar os principais conceitos da linguagem através de exercícios e pequenos desafios.

## 📚 Objetivos

- Aprender lógica de programação
- Desenvolver boas práticas em Python
- Criar uma base sólida para projetos maiores
- Evoluir para desenvolvimento Back-end

## ▶ Como executar

Clone o projeto

```bash
git clone https://github.com/Levidevbr/Python-PowerUp.git
```

Entre na pasta

```bash
cd Python-PowerUp
```

Execute qualquer arquivo

```bash
python nome_do_arquivo.py
```
